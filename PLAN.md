# Cooking Plan

We eat 5 meals a week:
- **Monday:** 2 cup freezer
- **Tuesday:** 2 cup freezer
- **Wednesday:** 2 cup freezer
- **Thursday:** 1 cup freezer
- **Saturday:** 2 cup freezer

## Preparation Strategy
We cook once a week. Since we consume four 2-cup meals and one 1-cup meal weekly, we run a cycle to maintain stock:

| Week | Cook | Stock (2-cup / 1-cup) | Max Storage |
| :--- | :--- | :--- | :--- |
| W1 | 5 x 2-cup | 5 x 2-cup / 5 x 1-cup | [Max 7] |
| W2 | 5 x 2-cup | 6 x 2-cup / 4 x 1-cup | [Max 8] |
| W3 | 5 x 2-cup | 7 x 2-cup / 3 x 1-cup | [Max 9] |
| W4 | 5 x 2-cup | 8 x 2-cup / 2 x 1-cup | [Max 10] |
| W5 | 5 x 2-cup | 9 x 2-cup / 1 x 1-cup | [Max 11] |
| W6 | 5 x 1-cup | 5 x 2-cup / 5 x 1-cup | - |

**Storage Goal:** We need capacity for **12 x 2-cup** meals and **5 x 1-cup** meals.

## Cycling Meals
- **Monday:** Pasta + Pasta Sauce + Topping
- **Tuesday:** Curry + Rice + Topping
- **Wednesday:** Stir-fry + Noodles + Topping
- **Thursday:** Dumplings + Veggies + Topping
- **Saturday:** Meat and veggie mix + Bulgur/Couscous + Topping
